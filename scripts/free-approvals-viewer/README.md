# Free Approvals Viewer (basic)

This is a very small “free preview” script from Wallet Guard.

## Run

```bash
npm install
node index.js 0x0000000000000000000000000000000000000000
```

## Notes

- This script is intentionally limited (MVP). It does **not** generate risk scoring or recommendations.
- For the full report with actions (REVOKE NOW / REVIEW / OK), use the Telegram bot.
